# Flagship Media Colors Design

**Date:** 2026-07-19
**Status:** Approved for implementation
**Branch:** `codex/stt-footer-layered-reveal`

## Objective

Differentiate the adjacent Call Agent and ConvoAI media stages with calm, editorial color fields drawn from the approved reference while preserving the existing flagship composition and interactions.

## Color Direction

- Call Agent uses warm ivory yellow `#e8ddbb`.
- ConvoAI uses cool mist blue `#dce9ef`.
- Both stages use solid colors only. No gradient or texture is introduced.
- The light, muted pair should separate the products without competing with their embedded media.

## Locked Behavior

- Existing media assets and positioning
- 20px media radius
- Desktop hover expansion, opacity, and focus behavior
- Mobile stacked layout
- Call Agent internal navigation and ConvoAI new-tab behavior

## Validation

- Computed media backgrounds match the approved RGB values.
- Both media backgrounds have no background image.
- Desktop focus motion still shifts between the two projects.
- Mobile cards remain stacked without transforms or horizontal overflow.
