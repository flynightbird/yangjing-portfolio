# Local portfolio fonts

The Xuelang case study uses three MiSans webfont weights supplied by the
portfolio owner in `/Users/admin/Desktop/MiSans.zip`:

- `MiSans/woff2/MiSans-Regular.woff2` -> `misans-regular.woff2` (400)
- `MiSans/woff2/MiSans-Semibold.woff2` -> `misans-semibold.woff2` (600)
- `MiSans/woff2/MiSans-Bold.woff2` -> `misans-bold.woff2` (700)

Geist Mono is installed through `@fontsource/geist-mono` and used for case
metadata, chapter indices, experiment labels, and metrics.
